# Learning Java 23

A comprehensive, hands-on guide to learning **Java 23** — from the absolute basics to advanced topics — using structured modules with examples and exercises.

## 📚 Modules

| # | Module | Topics |
|---|--------|--------|
| 01 | [Introduction to Java & Setup](modules/01-introduction-and-setup.md) | What is Java, JDK/JVM/JRE, installing Java 23, Hello World |
| 02 | [Variables, Data Types & Operators](modules/02-variables-datatypes-operators.md) | Primitive types, `var`, operators, type casting |
| 03 | [Control Flow](modules/03-control-flow.md) | if/else, switch expressions, for/while/do-while loops |
| 04 | [Object-Oriented Programming](modules/04-oop-classes-and-objects.md) | Classes, objects, constructors, encapsulation |
| 05 | [Inheritance & Polymorphism](modules/05-inheritance-and-polymorphism.md) | extends, method overriding, super, polymorphism |
| 06 | [Interfaces & Abstract Classes](modules/06-interfaces-and-abstract-classes.md) | interface, abstract class, default methods, sealed interfaces |
| 07 | [Collections Framework](modules/07-collections-framework.md) | List, Set, Map, Queue, iterators |
| 08 | [Generics](modules/08-generics.md) | Generic classes, methods, bounded type parameters, wildcards |
| 09 | [Exception Handling](modules/09-exception-handling.md) | try/catch/finally, custom exceptions, multi-catch |
| 10 | [Functional Programming](modules/10-functional-programming.md) | Lambdas, method references, Stream API, Optional |
| 11 | [Java I/O & NIO](modules/11-io-and-nio.md) | File I/O, Readers/Writers, NIO.2, Path API |

## 🚀 Getting Started

1. Install [Java 23](https://adoptium.net/) (or use [SDKMAN!](https://sdkman.io/))
2. Start with [Module 01](modules/01-introduction-and-setup.md)
3. Read the theory, run the examples, and complete the exercises at the end of each module

## 🛠 Prerequisites

- A computer running Windows, macOS, or Linux
- A text editor or IDE (VS Code, IntelliJ IDEA, Eclipse)
- Curiosity 🙂
